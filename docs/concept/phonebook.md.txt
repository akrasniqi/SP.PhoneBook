# **PhoneBookApp as SharePoint HostedApps**

======================

## Projekterstellung
* Die Repositorys erstelle ich in Git Hub
* Das Projekt wird auf Visual Studio angelegt, als SharePoint Hosted Apps. Die Standardstruktur werde ich beibehalten.
* Projektstruktur: Unterordner: docs, assets, src und eine .gitignore datei.
1. Unter Punkt Pages -> phonebook.aspx als zentraler Einstiegspunkt der App
2. Unter Punkt Content Ordner namens css -> phonebook.css hinzufügen
3. Unter Punkt Scripts Ordner namens js -> phonebook.js hinzufügen
* Sitecollection -> [Sitecollection für die Phonebookapp](https://heroesinside.sharepoint.com/sites/ak-dev)
